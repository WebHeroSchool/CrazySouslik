Мои навыки

Я постоянно стремлюсь к получению новых знаний, изучению современных технологий и методик работы. И поэтому мой скромный арсенал постоянно пополняется полезными инструментами.

  1. HTML 5

    Применяю адаптивную, кроссбраузерную верстку, с использование БЭМ-методологии и семантики.

  2. CSS 3

    Именно от CSS зависит внешний вид страницы, поэтому CSS я уделяю огромное внимание, стараясь устранить префиксы и перенаследование. При верстке пользуюсь Flexbox, Grid.

  3. JavaScript ES5/ES6

    Для создания интерактивности я использую javascript, стараясь писать чистый, легко поддерживаемый код, без лишнего дублирования

  4. Git

    Командная работа - очень важный элемент разработки, и контроль версий необходимый инструмент в работе над проектом. Поэтому для себя я избрал Git, как один из самых удобных инструментов контроля и организации работы.

  5. Gulp

    При сборке проектов изспользую надежный сборщик, который позволяет получить валидный, транспилированный JS-код, произвести обработку CSS-файлов, оптимизацию изображений и многое другое.

  6. PostCSS

    Надежный помошник в моей работе, который многократно ускоряет процесс кодинга.

  7. SASS/SCSS

    SASS/SCSS - удобный, понятный и широко используемый инструмент.
